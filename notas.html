Utilizar envolope com abreviaçao para inserir as tags:

CODE = deixar todas as letrascom o mesmo espaçamento e tamanho

PRE = serve para organizar um codigo escrito no html

Q = colocar aspas nas palavras

ABBR = aparece uma informação escrita no código ao passar o mouse encima da palavra

blockquote = referencia um pouco a direita

TYPE = Para definir o tipo de lista, letra, numero, letra maiscula / star = define o

ponto de partida, vai dar sequencia depois do 3 por exemplo

OL = lista ordenada: opções 1, A,a I,i / UL: lista não ordenadas: opcões disc, circle, square

A = colocar link no codigo (precisa envelopar para usar essa tag) > target="_blank" rel="external" para não fechar sua aba ao clicar no link

rel="next" ir para proxima pagina / rel="previous" voltar para pagina anterior / target="_self" abre na mesma pagina / ../: volta para pasta anterior

Picture para colocar imagens > source media: para definir o tamanho da imagem de acordo com o aparelho do usuario.

Regra de compatibilidade html e css: @charset "UTF-8" > colocar no CSS

Inova

laft = esquerda / right = direita / height = altura / top = principal / width = largura / padding = preenchimento / margin = margem

background-attachment: fixed = anexo de fundo: fixo Google fonts, obter codigo de vatios tipos de fontes Dafont : para baixar fontes diferenciadas Colorzila: Copiar fonts dos sites Extensão fonts ninja = para identificar as fontes do site
